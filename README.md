# game-of-life-cljs

[Conway's Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life) implemented
in ClojureScript.

* Supports multiple tilings.
* Share the starting state using a link.

Future Scope

- [ ] Infinite scroll

## Development
```shell
bun i
bun dev
```

This will start up a `nREPL` server locally.
